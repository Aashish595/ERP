ERP full hybrid backend

server/ is the complete Express 5 + TypeScript ERP backend.
ai-service/ is the private FastAPI AI microservice.

Express source architecture:
- 24 TypeScript model files / 52 database model definitions
- 20 repository files
- 26 service files
- 29 controller files
- 20 validator files
- 29 route files
- shared core, auth middleware, caching, uploads, errors and configuration
- complete PostgreSQL schema and indexes
- 255-route compatibility contract from the original FastAPI backend

The ZIP intentionally excludes node_modules, generated dist files and the old
Python ERP backend. Install dependencies using npm install. FastAPI contains only
AI endpoints and has no ERP table ownership.
